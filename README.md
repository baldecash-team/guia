# Guia BaldeCash

Documentacion, reportes de ciclos y recursos del equipo BaldeCash.

## Estructura

```
/
├── index.html          # Pagina principal
├── resumen/            # Reportes de ciclos semanales
│   ├── ciclo-27.html
│   ├── ciclo-28.html   (proximamente)
│   └── ...
└── README.md
```

## GitHub Pages

Este sitio esta desplegado en: https://baldecash-team.github.io/guia/

## Agregar nuevo ciclo

1. Crear archivo `resumen/ciclo-XX.html`
2. Actualizar `index.html` con el nuevo enlace
3. Commit y push a main
